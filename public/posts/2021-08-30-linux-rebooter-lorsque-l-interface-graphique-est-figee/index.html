Linux, rebooter lorsque l'interface graphique est figée

 2021-08-30

Si les ctrl alt F1 (etc…) ne permettent pas de basculer sur une console texte pour tuer les processus récalcitrants, il est possible de rebooter en utilisant une commande magique

alt Impr (syst) r e i s u b

Source : https://fr.wikipedia.org/wiki/Magic_SysRq_key
