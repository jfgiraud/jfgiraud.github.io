bash, lecture ligne à ligne et initialisation variables...

 2012-08-07

Dans la lignée du post précédent, il est possible de lire ligne à ligne et d’initialiser des variables directement pour des fichiers bien structurés.

Dans le cas d’ /etc/password : $ while IFS=: read user pass uid gid full home shell

do
printf “$full – Pseudo : $user UID : $uid GID : $gid Home : $home Shell : $shell\n”
done < /etc/passwd root – Pseudo : root UID : 0 GID : 0 Home : /root Shell : /bin/bash daemon – Pseudo : daemon UID : 1 GID : 1 Home : /usr/sbin Shell : /bin/sh bin – Pseudo : bin UID : 2 GID : 2 Home : /bin Shell : /bin/sh sys – Pseudo : sys UID : 3 GID : 3 Home : /dev Shell : /bin/sh
