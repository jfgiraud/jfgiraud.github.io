--- layout: post title: python, marre des accents en java : remplacez les par de l'unicode ! --- 


$ cat tounicode.py
#!/usr/bin/python

import re
import sys

s = sys.argv[1].decode('utf8')

print re.sub('\\\\x', '\\u00', re.sub('\'$', '', re.sub('^u\'', '', repr(s))))

$ python tounicode.py "salut beauté"
salut beaut\u00e9
